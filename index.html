🚀 견적 생성 기본 흐름
1POL / POD 검색 — 항구명 또는 코드로 검색, CY/DOOR 선택
2컨테이너 선택 — 타입 선택 후 수량 입력
31구간(선적지) → 2구간(해상운임) → 3구간(도착지) 구간별 원가·판가 입력
4FAK 조회 — 2구간 FAK 버튼 → 선사·SCAC 검색 → 카드 클릭으로 자동 입력
5✨ 견적서 생성 — 좌측 하단 플로팅 버튼 또는 입력 패널 하단 버튼
⚡ 주요 기능 요약
📥 FAK 조회
2구간 OFT 항목 옆 버튼. 선사·SCAC 검색 후 카드 클릭 → 원가 자동 입력
↻ 환율 조회
환율 섹션 버튼 클릭 → 실시간 USD/KRW 자동 입력
🖨 인쇄 / PDF
견적서 생성 후 하단 인쇄 버튼 → 대상: PDF로 저장 선택
🗂 견적 저장
견적서 하단 저장 버튼 → History 탭에 누적 보관
↩ 불러오기
History 탭 또는 입력 패널 상단 버튼 → 과거 견적 복원 후 수정
✦ AI 인사이트
Claude API Key 입력 시 화주 브리핑·수익 분석 자동 생성

견적서 탭 → 수익 시뮬레이션 탭으로 이동 (내부용)
